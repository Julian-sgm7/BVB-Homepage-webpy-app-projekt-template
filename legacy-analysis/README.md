# Legacy-Analyse: Quellcode-Basis

Dieses Verzeichnis enthält den extrahierten Legacy-Quellcode aus `upload/Projektarbeit.zip`.

## Aufbau
- `legacy-analysis/Projektarbeit/WebContent/` – Web-Anwendung mit PHP, CSS und Bildern
- `legacy-analysis/Projektarbeit/WebContent/controller/` – Controller-Logik
- `legacy-analysis/Projektarbeit/WebContent/model/` – Geschäftslogik
- `legacy-analysis/Projektarbeit/WebContent/view/` – Formular- und Ergebnisdarstellung

## Zweck
- Der hochgeladene Code dient als Ausgangsbasis für die weitere Repo-Entwicklung.
- Hier liegen die funktionalen Bestandteile der vorhandenen BVB-Homepage.
- Diese Struktur ist als Legacy-Analyse verfügbar, um den Übergang in eine moderne Repo-Architektur zu dokumentieren.
